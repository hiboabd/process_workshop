You receive the name of a city as a string, and you need to return a string that shows how many times each letter shows up in the string by using an asterisk.



Input | output

```
H       |    “h: *”
I       |    “I: *”
Hi      |    “h: *, i: *”
Hii     |  “h: *, i: **”
good    |   “g: *, o: **, d:*”
```
